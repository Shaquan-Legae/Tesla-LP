Tesla Clone Project
    A responsive, high-fidelity recreation of the Tesla homepage built with HTML5 and CSS3. 

    This project focuses on clean layout structures, interactive components, and mobile-first responsiveness.

Features:
    Dynamic Navigation: A sticky navbar with a functional hamburger menu for mobile devices.
    Hero Section: High-impact banners featuring current Tesla models and financing offers.Responsive Grid: Custom-styled product cards for vehicles and energy solutions that adapt from a 3-column desktop layout to a 1-column mobile stack.
    Offer Modules: Dedicated sections for promotional offers and community incentives.
    Interactive UI: Hover effects, transition animations, and a floating action button (FAB) for chat support.

Tech Stack:
    HTML5: Semantic structure.
    CSS3: Flexbox, Grid, Media Queries, and custom variables.
